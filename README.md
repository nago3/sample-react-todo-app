# React Todo App sample

## Commands

### setup

```
npx create-react-app@5.0.1 sample-5.0.1 --template typescript
```

### start

`npm start`

### import

`npm install ...`

#### test

```
npm install --save-dev \
  @testing-library/react@14 \
  @testing-library/jest-dom@5 \
  @testing-library/user-event@14
```
